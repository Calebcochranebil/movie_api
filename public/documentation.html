MyFlix Movies catalog

Objective

The objective of this application is to develop the server-side infrastructure for a web-based "movies" application, which will give users a comprehensive insight into diverse movies, directors, and genres. Alongside user authentication and profile maintenance capabilities, the application shall empower users to curate a catalog of their preferred movies.

Request URL Response
GET /movies Returns a list of all movies in the database.
GET /movies/:Title Returns details about a single movie, by title.
GET /movies/genres/:Name Returns details about a genre, by name.
POST /users Allows new users to register.
GET /users/:Username Returns data about a user, by username.
PUT /users/:Username Allows users to update their user information (username, password, email, date of birth).
POST /users/:Username/movies/:MovieID Allows users to add a movie to their list of favorites.
DELETE /users/:Username/movies/:MovieID Allows users to remove a movie from their list of favorites.
